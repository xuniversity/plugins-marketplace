# agent-team-vben-rules

团队协作与 Vben 前端规范插件，安装后可同时获得以下 2 个技能：

- `agent-team`
- `vben-component-rules`

## Skill 路径

- `skills/agent-team/SKILL.md`
- `skills/vben-component-rules/SKILL.md`

## 适用说明

- `agent-team`：适合需要 Orchestrator + 多角色子 Agent 协作的任务。
- `vben-component-rules`：适合遵循项目内 Vben 模板、共享组件、Swagger 约束的前端仓库。
